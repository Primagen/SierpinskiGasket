Sierpinski Gasket
==================

Creates a Sierpinski Gasket using GL_TRIANGLES to renders.

Made using:
  - OpenGL (>3.1)
  - GLEW (cross-platform loading GL extensions)
  - GLFW (window activity)
  - GLM  (vector point math)
  
Made with the help from Edward Angel's book:
  Interactive Computer Graphics - A Top-Down Approach with Shader-Based OpenGL, 6th Edition
